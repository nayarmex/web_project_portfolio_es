🚀 Descripción del Proyecto
Este portafolio no es solo una página de contacto; es un entorno diseñado para presentar mis trabajos destacados de forma visualmente atractiva. Incluye secciones dedicadas a:

Presentación Profesional: Mi perfil, experiencia y enlaces directos a mi CV.

Proyectos Destacados: Una galería que incluye aplicaciones como "My Sushi" y "My own Collage", detallando su funcionalidad y enlaces a código fuente.

Stack Tecnológico: Una visualización de las herramientas que domino.

🛠 Tecnologías y Técnicas Utilizadas
Frontend & Diseño
HTML5 & CSS3 Avanzado: Uso extensivo de Flexbox y Grid Layout para una estructura robusta.

Diseño Responsivo (Responsive Design): Implementación de Media Queries y la función clamp() para asegurar que la experiencia sea perfecta desde móviles de 320px hasta monitores de gran escala.

Metodología BEM (Block Element Modifier): Organización de clases para un código CSS escalable y fácil de mantener.

Efectos Visuales con Código:

Uso de Radial Gradients combinados con calc() para fondos dinámicos que mantienen la armonía visual en pantallas anchas.

Superposición de elementos mediante Position Absolute para crear efectos de oscurecimiento (overlays) en tarjetas de proyectos.

Filtros y efectos de cristal (Backdrop-filter: blur) para una estética moderna y profesional.

Herramientas de Desarrollo
Control de Versiones: Gestión profesional con Git y GitHub, incluyendo configuración de llaves SSH y sincronización de autoría local/remota.

Calidad de Código: Configuración de .editorconfig y .prettierignore para mantener la consistencia en el desarrollo.

📂 Estructura del Repositorio
index.html: Estructura principal del sitio.

/styles: Archivos CSS organizados por componentes.

/images & /icons: Recursos visuales optimizados.

.editorconfig & .gitignore: Configuración de entorno y exclusión de archivos de sistema.
